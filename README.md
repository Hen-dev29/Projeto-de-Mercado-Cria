# 🛒 Mercado Cria

## 📌 Sobre

Esse projeto é um site simples de mercado com sistema de carrinho de compras.

A ideia foi permitir adicionar produtos e calcular automaticamente o valor total da compra.

---

## 🚀 O que tem no projeto

- Adição de produtos ao carrinho  
- Cálculo automático do total  
- Interface simples e organizada  
- Uso de componentes do Bootstrap  

---

## 🛠️ Tecnologias usadas

- HTML  
- CSS  
- JavaScript  
- Bootstrap  

---

## 📚 O que eu pratiquei

- Lógica com JavaScript  
- Manipulação de elementos na página  
- Cálculo de valores  
- Uso de Bootstrap para layout  

---

## ▶️ Como abrir

Basta abrir o arquivo `index.html` no navegador.
